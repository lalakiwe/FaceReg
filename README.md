FaceReg
=======

A simple face recognition program which is implemented by QML and OpenCV.

**Develop environment**

 - Windows 7 64bit
	 - Visual Studio 2012
	 - QT 5.3.2 
	 - Qt Creator 3.2.2
	 - OpenCV 2.4.10
 - Ubuntu 14.04 64bit
	 - gcc 4.8.2
         - QT 5.3.2 (gcc 4.6.1)
	 - Qt Creator 3.2.2
	 - OpenCV 2.4.10
